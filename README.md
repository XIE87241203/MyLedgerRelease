# 我的账本 - 您的专属财务管家

`我的账本` 是一款简洁而强大的个人记账应用，旨在帮助您轻松管理日常收支，掌握财务状况，并有效实现预算目标。本应用**纯本地运行，无需联网，绝对保障您的数据隐私**。通过直观的用户界面和丰富的功能，让记账变得前所未有的简单。

## ✨ 主要功能

*   **收支记录**：
    *   **收入记录**：详细记录每一笔收入的来源、金额和时间，让您的财富增长一目了然。
    *   **支出记录**：精确追踪每一笔开销的去向，帮助您分析消费习惯，发现潜在的节约空间。
*   **预算管理**：
    *   灵活设置各类别的月度或年度预算，实时监控消费进度，有效控制不必要的开支，助您达成财务目标。
*   **多账本支持**：
    *   创建并管理多个独立的账本，无论是个人消费、家庭开支还是旅行基金，都能清晰划分，互不干扰。
*   **数据备份与恢复**：
    *   提供安全可靠的数据备份功能，确保您的每一笔财务记录都万无一失。同时支持便捷的数据恢复，让您无需担心数据丢失。
*   **自定义分类**：
    *   根据个人需求，自由添加、编辑和管理收支分类，让您的记账更加个性化和精细化。

## 🚀 为什么选择我的账本？

*   **简洁易用**：直观的用户界面和流畅的操作体验，让记账不再是负担。
*   **功能全面**：覆盖日常记账、预算控制、多账本管理等核心需求。
*   **数据安全**：提供数据备份与恢复机制，保障您的财务信息安全。
*   **高度自由**：自定义分类功能，满足您的个性化记账需求。
*   **绝对隐私**：纯本地运行，无需联网，您的所有财务数据都将安全地保存在您的设备上，绝不上传，充分保障个人隐私。
